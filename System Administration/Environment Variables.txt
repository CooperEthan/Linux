ENVIRONMENT VARIABLES

- To view all env variables
	- printenv or env
- To view ONE env variabel
	- echo $SHELL
- To set the environement variables
	- export TEST=1
	- echo $TEST
- To set env variable permanently
	- vi .bashrc
	- TEST ='123'
	- export TEST
- To set global env variable permanently
	- vi /etc/profile or /etc/bashrc
	- TEST ='123'
	- export TEST
