LINUX KERNEL

Kernel is a interface between hardware and software

	USER
	 |
      Application
	 |
       Shell
         |
       Kernel
         |
      Hardware ( Cpu, Memory, HD )

Application + shell = Software
Shell + Kernel = Operating system

TYPES OF LINUX SHELLS

-	Gnome
-	KDE
-	sh  --> Born shell
-	bash --> Born Again shell  --> used in Linux environment
-	csh and tchs 
-	ksh  ( compatable with sh and bash )  --> it is used Solaris environment
